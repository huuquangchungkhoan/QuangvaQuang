# Custom Dashboards Project

4 custom dashboards sử dụng MUI Dashboard template.

## 🚀 Quick Start

```bash
cd frontend
npm start
```

Opens http://localhost:3000

## 📊 Dashboards

| Dashboard | Route | Description |
|-----------|-------|-------------|
| **Thị trường** | `/thi-truong` | Market dashboard |
| **Quỹ mở** | `/quy-mo` | Open Funds |
| **So sánh cổ phiếu** | `/so-sanh` | Stock Comparison |
| **Ratios** | `/ratios` | Financial Ratios |

## 📦 Tech Stack

- React 18 (create-react-app)
- Material-UI v6
- MUI X (Charts, Data Grid, Date Pickers, Tree View)
- React Router v6
- Emotion

## 📁 Structure

```
frontend/
```

## ✨ Features

- ✅ 4 separate dashboards
- ✅ Sidebar navigation
- ✅ Responsive design

## 🎨 Customize

Edit content in each dashboard's `MainGrid.js`:
- `src/thi-truong/components/MainGrid.js`
- `src/quy-mo/components/MainGrid.js`
- `src/so-sanh/components/MainGrid.js`
- `src/ratios/components/MainGrid.js`
